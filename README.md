# Immersive Portfolio (Paris Experience)

This project is a web experience built with:
- **HTML + CSS** (page layout + design)
- **JavaScript** (logic, interactions)
- **Vite** (dev server + fast builds)
- **Three.js** (3D scene)
- *(Optional later)* **React** (UI/content management) and **GSAP** (smooth animations)

> Important: This project **will NOT work by double-clicking the HTML file** (file://).  
> You must run it using the dev server.

---

## 1) Requirements (install these first)

### Install Node.js
- Download and install **Node.js (LTS)** from the official Node website.
- After install, verify:

```bash
node -v
npm -v


### Install dependencies

```bash
npm install
```

### Run the dev server

```bash
npm run dev
```

Open the correct page (VERY IMPORTANT)

http://localhost:5173/src/paris-immersive.html
